会员套餐消费模式：
强制金额消费：无论服务消费模式如何设定，企业在升级、续费会员时，必须使用金额进行消费。
说明：会员服务作为招聘网主要收入来源，您可能需要差异化地设定来切合运营需求，该设置目的是让站长在设定积分消费模式的情况下不影响主体营收。
